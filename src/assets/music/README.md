# Medieval Music Loops

- `medieval-ensemble-loop.ogg`: active in-game mix. The recorder carries the
  melody, Folk Harp plays modal broken chords, and Bowed Psaltery supplies a
  sustained string line.

- `medieval-recorder-loop.ogg`: original 32-bar A Dorian composition, 84 BPM,
  preserved as the recorder-only alternate.
- Instrument sources: VCSL Baroque Alto Recorder, Folk Harp, and Bowed
  Psaltery samples by Versilian Studios LLC.
- Source license: CC0 (public domain). VCSL permits commercial and
  non-commercial use without royalties or attribution.
- Library page: https://versilian-studios.com/vcsl/

Both assets are stereo Opus in Ogg containers at 48 kHz and loop at the exact
32-bar boundary.
